whatinfo
========

whatinfo - display info for file's or command's

Example
=======

./whatinfo wget

./whatinfo /etc/httpd.conf

or you can move it " mv whatinfo /usr/bin" to fast use


whatinfo wget

or

whatinfo /etc/httpd.conf


SYSTEM REQUIREMENTS
========

Red Hat - Centos - Fedora - RPM Packages
